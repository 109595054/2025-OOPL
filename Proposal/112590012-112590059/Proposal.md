# Game Introduction

遊戲共有四種角色，每一種角色的特殊技能都不同，玩家可利用其中一種角色進入遊戲，每次進入遊戲時會有多條的爬塔路線，
玩家可自行選擇經由哪條路徑向上破關，
與敵人戰鬥時，系統會抽取牌庫的牌至手上，玩家要利用有限的花費進行進攻、防守、施展技能等，
當玩家血量減至0時，便會死亡，玩家須重新選角爬塔，只有遊戲中所碰到過的卡片、遺物會記錄於百科大全，
當擊殺敵人後，會得到金幣、新卡、遺物等道具，
金幣可於商店購買新道具
新卡可以提升排組整體強度
